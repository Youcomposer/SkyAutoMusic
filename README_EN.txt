Music AutoPlayer for PC
Author: Youcomposer and OstbornTV
Version: 1.0
Date: 2024-09-15

Description
This Python tool simulates keystrokes on your computer to automatically play musical notes. It supports both .txt and .json files to define the notes to be played and allows adjustment of playback speed.

Features
File Selection: Choose a .txt or .json file that contains the notes.
Speed Adjustment: Select the playback speed after choosing the file (slow, medium, fast).
Automated Input: Simulates typing of notes on your computer.
Requirements
Python 3.x
pydirectinput: For simulating keystrokes.
tkinter: For the graphical user interface.
Installation
Github Link

Instructions
Ensure you have Python installed on your system.
Download the zip file (SkyAutoMusic.zip) from the repository.
Place your music files in the "Songs" directory.
Run the script by double-clicking SkyAutoMusic.bat.
Follow the on-screen instructions to choose music and start playing.
Enjoy your musical journey in Sky: Children of the Light!
