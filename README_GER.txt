# Music AutoPlayer for PC

**Author:** Youcomposer and OstbornTV
**Version:** 1.0  
**Date:** 2024-09-15

## Beschreibung

Dieses Python-Tool simuliert Tasteneingaben auf Ihrem Computer, um Musiknoten automatisch abzuspielen. Es unterstützt sowohl `.txt`- als auch `.json`-Dateien zur Definition der zu spielenden Noten und ermöglicht die Anpassung der Wiedergabegeschwindigkeit.

## Funktionen

- **Dateiauswahl**: Wählen Sie eine `.txt`- oder `.json`-Datei aus, die die Noten enthält.
- **Geschwindigkeitsanpassung**: Wählen Sie die Wiedergabegeschwindigkeit aus, nachdem die Datei ausgewählt wurde (langsam, mittel, schnell).
- **Automatisierte Eingaben**: Simuliert die Eingabe von Noten auf Ihrem Computer.

## Voraussetzungen

- Python 3.x
- `pydirectinput`: Zum Simulieren von Tasteneingaben.
- `tkinter`: Für die grafische Benutzeroberfläche.

## Installation

Github Link

##

1. Ensure you have Python installed on your system.
2. Download the zip file (SkyAutoMusic.zip) from the repository.
3. Place your music files in the "Songs" directory.
4. Run the script by double-clicking SkyAutoMusic.bat
5. Follow the on-screen instructions to choose music and start playing.
